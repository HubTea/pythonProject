실행 방법
main.py를 실행



필요 라이브러리
PyOpenGL 3.1.5(http://pyopengl.sourceforge.net/)
PyQt5 5.15.4(https://pypi.org/project/PyQt5/)
Numpy(https://numpy.org/)




매뉴얼

마우스 왼쪽 클릭 : 면 선택
마우스 오른쪽 클릭 : 면의 색 지정
마우스 가운데 클릭 : 물체 전체 선택

키보드
왼쪽 방향키 : 카메라를 왼쪽으로 이동
오른쪽 방향키 : 카메라를 오른쪽으로 이동
위쪽 방향키 : 카메라를 위로 이동
아래쪽 방향키 : 카메라를 아래로 이동

W : 위쪽 바라봄
S : 아래쪽 바라봄
A : 왼쪽을 바라봄
D : 오른쪽을 바라봄
Q : 카메라를 반시계방향으로 회전
E : 카메라를 시계방향으로 회전

R : 화면 확대
F : 화면 축소

T, G, Y, H : 화면 가운데의 하늘색 점을 중심으로 회전

1 : 선택 전부 해제
2 : 아무것도 선택 안 함 모드
3 : 면 선택 모드

4 : 한번에 하나의 면만 선택하는 모드
5 : 이어서 선택하는 모드
6 : 선택 해제 모드

7 : x축 선택
8 : y축 선택
9 : z축 선택

- : 복제. 기존 물체와 연결되어 있음
= : 복제. 기존 물체와 연결되어 있지 않음

delete : 선택된 면 삭제

I : 선택된 면 뒤집음

F1 : mesh.3d파일로 저장
F2 : mesh.3d파일 불러오기

Z : 평행이동
X : 회전
C : 크기 변경
Z, X, C 누른 뒤 마우스를 화면 중앙에서 가깝게 하거나 멀어지게 해서 조절
조절 끝나면 눌렀던 키를 다시 누르면 완료

V : 와이어프레임 모드. ON/OFF

M : Catmull-Clark Subdivision 한 단계 증가
N : Catmull-Clark Sbudivision 한 단계 감소

콘솔 창
rgb r g b : 색 설정. ex) rgb 255 0 0
